## ✅ التحقق من الإضافة

بعد إضافة الترجمات، استخدمها في مكوناتك:

### مثال - في مكون Chatbot:
```typescript
import { useLanguage } from "@/contexts/LanguageContext";

export function Chatbot() {
  const { t } = useLanguage();

  return (
    <div>
      {/* العنوان سيظهر بالعربية أو الإنجليزية تلقائياً */}
      <h1>{t("chatbotTitle")}</h1>
      <p>{t("howCanIHelp")}</p>
      <button>{t("sendMessage")}</button>
    </div>
  );
}
```

### مثال - في مكون Pricing:
```typescript
export function Pricing() {
  const { t, formatCurrency } = useLanguage();

  return (
    <div>
      <h2>{t("fixedPricing")}</h2>
      <p>{t("noAdditionalFees")}</p>
      <span>{formatCurrency(100)}</span>
    </div>
  );
}
```
