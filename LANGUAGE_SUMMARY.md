## 🎓 أمثلة عملية سريعة

### استخدام بسيط:
```typescript
// في أي مكون
const { t } = useLanguage();

<button>{t("bookNow")}</button>
// ✅ تظهر: "احجز الآن" (عربي) أو "Book Now" (إنجليزي)
```

### مع التنسيق:
```typescript
const { t, formatCurrency } = useLanguage();

<span>{formatCurrency(150)}</span>
// ✅ تظهر: "ج.م ١٥٠" (عربي) أو "150 EGP" (إنجليزي)
```

### مع الاتجاه:
```typescript
const { isRTL } = useLanguage();

<div dir={isRTL ? "rtl" : "ltr"}>
  {/* المحتوى ينقلب تلقائياً */}
</div>
```

### تبديل اللغة:
```typescript
const { language, setLanguage } = useLanguage();

<button onClick={() => setLanguage(language === "ar" ? "en" : "ar")}>
  تبديل للإنجليزية / Switch to English
</button>
```

## 📞 أسئلة شائعة

### س: هل يجب إضافة الترجمات الآن؟
**ج:** لا، النظام يعمل بدونها. أضفها عند تطوير الميزات الجديدة.

### س: كيف أعطل اللغة الحالية؟
**ج:** في LanguageContext.tsx غير `"ar"` إلى `"en"` في السطر الافتراضي.

### س: هل يمكن إضافة لغة ثالثة؟
**ج:** نعم، أضف كائن جديد في `translations` مع كل الترجمات.

### س: ماذا لو كانت الترجمة ناقصة؟
**ج:** تظهر نفس الـ key بدلاً من النص. أضفها في LanguageContext.
