# ✅ دليل اختبار واستخدام نظام اللغات

## 🎯 الهدف

التأكد من أن نظام اللغات يعمل بشكل صحيح في الموقع.

---

## 📋 قائمة الاختبار الشاملة

### الخطوة 1️⃣: فتح الموقع

```
✅ رابط: http://localhost:8080
✅ تحميل الصفحة الرئيسية
✅ الصفحة تحمل بسهولة
```

---

### الخطوة 2️⃣: التحقق من اللغة الافتراضية

```
📌 عند تحميل الموقع لأول مرة:
   - اللغة الافتراضية: العربية ✅
   - الاتجاه: من اليمين لليسار (RTL) ✅
   - النصوص: عربي واضح ✅
```

**ما الذي تتحقق منه:**
- [ ] عنوان الصفحة بالعربية
- [ ] القائمة باليمين (RTL)
- [ ] الأزرار محاذاة صحيحة
- [ ] الأيقونات في المكان الصحيح

---

### الخطوة 3️⃣: البحث عن زر تبديل اللغة

```
📍 موقع الزر:
   - أعلى الصفحة (Navbar)
   - الزاوية اليسرى أو اليمنى
   - رمز عالمي 🌐 مع نص "English" أو "عربي"
```

**ما الذي تتحقق منه:**
- [ ] وجود زر تبديل اللغة
- [ ] ظهور الرمز 🌐 بوضوح
- [ ] النص واضح ومقروء

---

### الخطوة 4️⃣: اضغط على زر تبديل اللغة

```
🔄 عند الضغط:
   ✅ اللغة تتبدل من العربية إلى الإنجليزية
   ✅ جميع النصوص تتحول للإنجليزية فوراً
   ✅ الاتجاه يتغير من RTL إلى LTR
   ✅ الأزرار تنقلب
   ✅ الهوامش والحشوة تنقلب
```

**ما الذي تتحقق منه:**
- [ ] التبديل فوري (بدون تأخير)
- [ ] جميع النصوص بالإنجليزية
- [ ] الاتجاه من اليسار لليمين (LTR)
- [ ] الأيقونات في المواضع الصحيحة

---

### الخطوة 5️⃣: التحقق من جودة الترجمة

#### الصفحة الرئيسية:
```
عنوان                  | العربية              | الإنجليزية
────────────────────────────────────────────────────────
العنوان الرئيسي        | ✅ نص عربي واضح    | ✅ واضح
الوصف                  | ✅ يحكي عن الخدمة   | ✅ نفس المعنى
رسالة المرحبة         | ✅ بالعربية الفصحى  | ✅ بالإنجليزية الاحترافية
```

#### صفحة الخدمات:
```
الخدمة                | العربية        | الإنجليزية
────────────────────────────────────────
التنظيف              | ✅ تنظيف       | ✅ Cleaning
السباكة              | ✅ سباكة       | ✅ Plumbing
الكهرباء             | ✅ كهربائي     | ✅ Electrical
```

---

### الخطوة 6️⃣: اضغط مرة أخرى للعودة للعربية

```
🔄 عند الضغط مرة أخرى:
   ✅ العودة للعربية فوراً
   ✅ جميع النصوص عربية
   ✅ الاتجاه RTL من جديد
   ✅ التصميم متطابق تماماً
```

---

### الخطوة 7️⃣: تحديث الصفحة مع الانتظار للغة

```
📌 السيناريو:
   1. اضغط على الإنجليزية
   2. لاحظ أن الموقع بالإنجليزية
   3. اضغط F5 (إعادة تحميل)
   4. ماذا يحدث؟
```

**النتيجة المتوقعة:**
```
✅ الموقع يحمل بالإنجليزية (تم حفظ التفضيل)
✅ لا عودة للعربية الافتراضية
✅ اللغة محفوظة في localStorage
```

---

### الخطوة 8️⃣: تحديث الصفحة مع الانتظار للعربية

```
📌 السيناريو:
   1. اضغط على العربية
   2. لاحظ أن الموقع بالعربية
   3. أغلق المتصفح وافتحه مرة أخرى
   4. ماذا يحدث؟
```

**النتيجة المتوقعة:**
```
✅ الموقع يحمل بالعربية (تم حفظ التفضيل)
✅ نفس اللغة عند إعادة فتح المتصفح
✅ حفظ دائم للتفضيلات
```

---

### الخطوة 9️⃣: اختبار صفحات أخرى


#### صفحة الملف الشخصي:
```

─────────────────────────────
البيانات الشخصية   | [ ] بالعربية والإنجليزية
الخيارات           | [ ] ترجمة صحيحة
الأزرار            | [ ] ترجمة صحيحة
الرسائل            | [ ] ترجمة دقيقة
```

#### صفحة لوحة التحكم:
```
العنصر              | اختبر
─────────────────────────────
جداول البيانات     | [ ] بالعربية والإنجليزية
الرسوم البيانية    | [ ] ترجمة المحاور
الإحصائيات         | [ ] أرقام واضحة
الرسائل            | [ ] ترجمة صحيحة
```

## 🔍 النقاط الحرجة للاختبار

### ✅ يجب أن تعمل:

```
1. تبديل اللغة فوري (< 100ms)
2. حفظ اللغة المختارة
3. RTL/LTR تلقائي
4. جميع النصوص مترجمة
5. الأزرار والروابط مترجمة
6. الرسائل والتنبيهات مترجمة
7. تنسيق الأسعار صحيح
8. الأيقونات في المواضع الصحيحة
```

### ❌ لا يجب أن يحدث:

```
1. ✋ نصوص عربية في وضع الإنجليزية
2. ✋ نصوص إنجليزية في وضع العربية
3. ✋ تأخير في التبديل
4. ✋ فقدان النصوص
5. ✋ اختلاط RTL/LTR
6. ✋ أيقونات في مواضع خاطئة
7. ✋ أخطاء في اللغة أو المعاني
```

## 🛠️ في حالة وجود مشاكل

### مشكلة 1: النصوص لا تتبدل

```
✅ الحل:
   1. افتح DevTools (F12)
   2. اذهب الـ Console
   3. تحقق من الأخطاء
   4. تأكد أن LanguageContext موجود
   5. أعد تحميل الصفحة
```

### مشكلة 2: اللغة لا تُحفظ

```
✅ الحل:
   1. افتح DevTools (F12)
   2. اذهب الـ Application
   3. شاهد localStorage
   4. ابحث عن "language" key
   5. تأكد أنها موجودة بقيمة "ar" أو "en"
```

### مشكلة 3: RTL/LTR لا يتغير

```
✅ الحل:
   1. افتح DevTools
   2. افحص العنصر <html>
   3. تحقق من dir attribute
   4. يجب أن يكون dir="rtl" أو dir="ltr"
   5. أعد تحميل الصفحة
```

### مشكلة 4: ترجمات ناقصة

```
✅ الحل:
   1. افتح LanguageContext.tsx
   2. ابحث عن الترجمة الناقصة
   3. أضفها في كلا الكائنات (ar و en)
   4. احفظ الملف
   5. أعد تحميل الصفحة (Ctrl+Shift+R)
```
